# SOKOMIRU（そこみる）

## アプリの概要
URL: https://www.sokomiru.com/

YouTube動画とセットでメモを記録・管理できるWebアプリケーションです。
1番の推しポイントは、**何度も見返したい動画の場面も一緒に記録**できる点です。
